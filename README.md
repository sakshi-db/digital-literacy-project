## 📌 Project Overview
As a selected Student Digital Ambassador, I have developed this portfolio to help my peers navigate the digital world safely and professionally. This repository contains a series of tasks mapping to the five modules of the Digital Literacy course, demonstrating skills in visual communication, professional networking, technical collaboration, and cybersecurity awareness.
